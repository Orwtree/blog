项目简介

ShineBrowser 是一款基于 C# 和 Chromium 内核的轻量级浏览器。 设计目标是提供简洁、快速、注重隐私的上网体验，去除冗余功能，让浏览器回归工具本质。

✅ 当前版本已完成基础导航、书签管理、隐私模式等核心功能，适合日常轻度使用。

核心特性

  • ⚡ 极速启动 — 无插件、无后台进程，冷启动时间 < 1.5 秒
  • 🔒 隐私优先 — 内置隐私模式，不保存历史记录、Cookie 和缓存
  • 📑 标签管理 — 支持多标签页、拖拽排序、快捷键切换
  • 🔖 书签系统 — 支持导入/导出 HTML 书签，快速访问常用网站
  • 🎨 简洁界面 — 无广告、无推荐内容，专注于内容本身

技术栈

  • 语言 — C# (.NET Framework 4.8 / .NET 6+)
  • UI 框架 — Windows Forms
  • 渲染引擎 — Chromium Embedded Framework (CEF)
  • 数据存储 — JSON 本地配置文件
